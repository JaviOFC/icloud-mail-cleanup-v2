### Integration Gaps

#### INT-01: TUI Pipeline Missing Feedback Signal (Minor)

**Location:** `tui/screens/pipeline.py:142`
**Issue:** `classify_single(msg, profiles, replied_conv_ids, now)` called without `feedback` argument. CLI `cmd_classify` loads feedback.db and passes `sender_feedback`. TUI pipeline silently drops it.
**Impact:** Second+ TUI pipeline runs don't benefit from prior user decisions. First run unaffected.
**Affected:** CLAS-02
**Fix:** 4 lines — load FeedbackStore, extract sender_feedback, pass to classify_single.

#### INT-02: TUI Execute Missing Protection Overrides (Minor)

**Location:** `tui/screens/execute.py:199`
**Issue:** `execute_deletions()` called without `protection_overrides`. CLI `cmd_execute` extracts overrides from session.
**Impact:** Protected emails user explicitly overrode in TUI review are silently skipped during TUI execution.
**Affected:** EXEC-03
**Fix:** 2 lines — extract overrides from `self.app.session.protection_overrides`, pass to `execute_deletions`.
