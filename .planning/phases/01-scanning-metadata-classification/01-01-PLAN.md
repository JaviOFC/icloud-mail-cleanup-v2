  <behavior>

    - Test: get_sender_stats() returns dict keyed by lowercase address with count, total_size, min_date, max_date
    - Test: get_sender_stats() normalizes email case (Friend@example.com and friend@example.com are same sender)
    - Test: get_sent_recipients() returns dict of lowercase addresses with send_count and last_sent_to from Sent mailbox only
    - Test: get_replied_conversation_ids() returns set of conversation_ids that have messages in Sent mailbox
    - Test: get_replied_conversation_ids() excludes conversation_id = 0
  </behavior>
  <action>
    Create `src/icloud_cleanup/scanner.py` with:

    1. Constants: `ENVELOPE_INDEX = Path.home() / "Library/Mail/V10/MailData/Envelope Index"`, `ICLOUD_UUID = "XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX"`.

    2. `open_db(path: Path | None = None) -> sqlite3.Connection`: Opens the DB in URI read-only mode (`file:{path}?mode=ro`). Uses ENVELOPE_INDEX as default. Sets `row_factory = sqlite3.Row`.

    3. `scan_messages(conn: sqlite3.Connection) -> list[Message]`: Executes the canonical bulk query from research (joining messages, mailboxes, addresses, subjects, message_global_data). Filters by `mb.url LIKE 'imap://{ICLOUD_UUID}/%'`. Returns list of Message dataclass instances. Uses COALESCE for nullable address/subject.

    4. `get_sender_stats(conn: sqlite3.Connection) -> dict[str, dict]`: GROUP BY LOWER(sender_address) across all iCloud mailboxes. Returns dict with keys: count, total_size, min_date, max_date, last_received. Excludes Sent/Drafts mailboxes.

    5. `get_sent_recipients(conn: sqlite3.Connection) -> dict[str, dict]`: Queries recipients table joined with messages from Sent mailbox. Returns dict keyed by LOWER(address) with times_sent_to and last_sent_to.

    6. `get_replied_conversation_ids(conn: sqlite3.Connection) -> set[int]`: Returns set of distinct conversation_ids from messages in Sent mailbox where conversation_id > 0.

    All functions accept a connection object (not path) for testability. The query for sent recipients uses: `messages JOIN mailboxes ON m.mailbox = mb.ROWID JOIN recipients r ON r.message = m.ROWID JOIN addresses a ON r.address = a.ROWID WHERE mb.url LIKE 'imap://{UUID}/Sent%'`.

    - 1 message in a non-iCloud mailbox (should be filtered out)
    - Messages with NULL sender and subject (test COALESCE)
    - Conversation IDs: some shared between INBOX and Sent (for reply detection)
