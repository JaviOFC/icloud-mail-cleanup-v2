       - `on_mount()` or `on_screen_resume()`: Read `self.app.report_data`, `self.app.classifications`, `self.app.messages`, `self.app.session`. Build cluster data for all tiers (not just Review -- user sees everything). Load existing session decisions to mark already-reviewed clusters.

       - `on_data_table_row_highlighted(event)`: Get cluster label from row key, find matching cluster data in report_data, get the Classification objects for that cluster, pass to ClusterDetailWidget.show_cluster()

       - `action_toggle_select()`: Delegate to ClusterListWidget.key_space
       - `action_approve_selected()`: For each selected cluster, record decision in ReviewSession as `{"action": "approve", "message_ids": [...], "timestamp": int(time.time())}`. Call `save_session()`. Then for each approved cluster, look up the primary sender and call `find_propagation_targets()`. If suggestions found, show toast via `self.notify(f"{total} similar emails found. Check Propagation tab.", title="Propagation", severity="information", timeout=5)`. Update cluster list to show approved status (strikethrough or dimmed).
       - `action_skip_selected()`: Same pattern but action="skip". No propagation check.

       **Auto-triage button:**
       - `on_button_pressed(event)` handler for #btn-triage: Run `auto_triage()` with current Review-tier classifications and sender_lookup. Show result summary via `self.notify()`. Auto-apply resolved decisions to session. Refresh cluster list to reflect changes (remove or dim auto-resolved clusters).

       **Inspect toggle:**
       - `action_toggle_inspect()`: Toggle inspect mode on ClusterDetailWidget. When active, individual emails are shown inline in the detail panel.

       **Session persistence:**
       - After EVERY decision (approve, skip, auto-triage), call `save_session(self.app.session, self.app.session_path)`. This ensures crash-safe resumability and interoperability with CLI review.
       - On mount, if session has existing decisions, mark those clusters as already-decided in the list (visual indicator).
